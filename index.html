Welcome to the greatest Tic Tac Toe Game Ever

Requirements

  1. A user should be able to click on different squares to make a move
  2. Every click will alternate between marking an `X` and `O`
  3. Upon marking of an individual cell, use JavaScript to add a class to each cell to display separate colors
  4. A cell should not be able to be replayed once marked
  5. Add a reset button that will clear the contents of the board
